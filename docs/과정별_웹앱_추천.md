## 10~13과정 추천 (아직 만들지 않음, 메모만)

- **10과정 — 마이크로티칭 동료 피드백 보드** (16~19차시)
  - 시연자별 방을 만들고 관찰자가 체크리스트(학생 참여 촉진 / 교사-학생-AI 상호작용 / 과정중심평가 4요소)로 평가 + 한줄 피드백 입력.
  - 시연 끝나면 시연자 화면에 관찰 결과가 레이더/막대로 자동 집계. 교사 대시보드에서 방 생성·마감.
  - 교안 근거: 전개④ 마이크로티칭(80분) + 전개⑤ 상호 피드백(40분), "실시간 과정중심평가 체크리스트 및 관찰 기록 앱" 준비물 명시.

- **11과정 — 가상 학급 데이터 생성기 + 분석 실습** (20~21차시)
  - 교안에 "데이터 생성기를 통해 준비된 익명화 원시 데이터" 준비물이 명시되어 있어 필요성 가장 높음.
  - 학년·과목·학생 수를 고르면 성취·참여·정서·행동 4차원 가상 데이터(오답 패턴, 로그, 관찰 메모 포함)를 생성 → CSV 다운로드(스프레드시트 실습용).
  - 함정 데이터 포함: 네트워크 오류로 참여 저조처럼 보이는 학생, 찍어서 정답률 높은 학생 등 → "AI 분석 vs 교사 해석" 비교 활동과 편향 검증 토론에 사용.
  - 보조 기능: 같은 데이터를 Solar에 분석시켜 'AI 분석 결과'를 보여주고, 교사가 빠진 맥락을 찾게 함.

- **12과정 — 환류 전략 작성 도우미** (22~24차시)
  - 학생 데이터 요약을 입력하면 존 해티 피드백 3단계(Feed-Up / Feed-Back / Feed-Forward) 틀로 AI가 피드백 초안 제안 → 교사가 다듬어 환류 전략 템플릿 완성 → PNG/인쇄.
  - SAFE 원리(사회정서) 체크 항목 포함. 동료 코칭 기록지 모드 추가 가능.
